##ОТЛИЧИЕ INNER JOIN И LEFT JOIN##


1. INNER JOIN - показывает только те записи, которые есть в обеих таблицах. Если у ученика нет профиля - он не попадет в результат.

LEFT JOIN - показывает всех учеников из первой таблицы (students), даже если у них нет профиля. Просто в колонках из второй таблицы будет пусто (NULL).
